# PIO-01 — Parallax Injection Operator

Adaptive system for introducing concepts at the right time to improve human understanding, decision-making, and learning under ambiguity.

## Core Idea
People struggle not because tasks are complex—but because information is introduced at the wrong time relative to their understanding.

PIO-01 aligns information exposure with cognitive readiness:
- stabilize → expand → transition

## Components
- Driftwood Layer (DL-01)
- Lateral Expansion (LE-01)
- Saddle Gating (SG-01)
- Cadence Budget (CB-01)
- Admissibility Thresholds (AT-01)
