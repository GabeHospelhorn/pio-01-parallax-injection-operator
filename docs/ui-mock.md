# PIO-01 — UI Mock Specification

## Layout

Three-lane system:

1. Primary Lane (core concept)
2. Peripheral Lane (driftwood)
3. Crossing Lane (saddle prompts)

---

## Screen Model

### Primary Panel
- active concept
- working summary
- frame toggles

### Peripheral Panel
- 1–3 driftwood cards
- optional engagement

### Crossing Panel
- saddle prompts
- branch history

---

## Example Interaction

### Step 1
User explores concept
→ no interruption

### Step 2
Driftwood appears
→ optional

### Step 3
User engages
→ alternate frame shown

### Step 4
System detects stability
→ saddle prompt appears

### Step 5
User explores new basin

---

## Key UX Principles

- no forced learning
- no interruption of flow
- reversible exploration
- visible structure

---

## Why This Matters

Most AI interfaces:
- collapse all cognition into a single thread

PIO:
- separates stabilization, expansion, and transition

---

## Summary

> This UI turns learning into navigation across conceptual space.
